Youtube Comment Sentiment Analysis

Developed a Python-based YouTube comment sentiment analysis project using NLTK for preprocessing, TF-IDF vectorization, scikit-learn for logistic regression classification, and VADER for sentiment scoring to categorize comments as positive, negative, or neutral. This NLP tool extracts actionable audience insights for content creators and event promoters. Highlights skills in ML pipelines, data processing, and real-time feedback analysis.
